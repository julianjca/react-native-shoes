# React Native Shoes

## How to run project
### Set up React Native Development ENV
https://reactnative.dev/docs/environment-setup

### Install packages
```bash
npm install
or
yarn install
```

### Install the dependencies for iOS
```bash
cd ios
pod install
```

### Run the project
```bash
npx react-native start
```

```bash
npx react-native run-ios
```
